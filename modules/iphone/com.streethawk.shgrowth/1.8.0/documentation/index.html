Growth Module

Description

StreetHawk Growth module lets your user share your application with friends. Also the feature can be extended to invite the new user to a deeplinked page inside your application. Create a Share button inside your application and call originateShareWithSourceSelection to generate the share url. StreetHawk SDK will then ask user ways to share the generated url. That is (email, post on Facebook etc). Below are the APIs which generates sharable URL. A scheme for deeplinking is required for StreetHawk growth.

Accessing the Growth Module

To access this module from JavaScript, you would do the following:

var shGrowth = require("com.streethawk.Growth");

The shGrowth variable is a reference to the Module object.

growth.function

  • Get growth url callback handler

    function growthShareCallback(e) { alert("Callback for growth share url: " + e.result); } //Get growth share url by callback shGrowth.originateShareWithCampaign("ShareViaFacebook", "Facebook", "cpc", "best chocolate chip cookie recipe", "facebookAdd", "tisample://homepage?recipe=chocolatechipcookie", "http://recipeApp.com/chocolatechipcookie", {shareurl: growthShareCallback}); //must use key "shareurl"

  • Share by predefined channels

    shGrowth.originateShareWithSourceSelection("ShareViaFacebook", "tisample://homepage?recipe=chocolatechipcookie", "http://recipeApp.com/chocolatechipcookie");
